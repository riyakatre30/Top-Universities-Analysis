# Top University Data Analysis

## Project Overview

This project performs **Exploratory Data Analysis (EDA)** on the **Top University Dataset** 
using Python. The objective is to analyze global university rankings and identify patterns 
based on educational quality, research performance, faculty quality, citations, publications, 
patents, and overall university scores. The analysis includes data cleaning, descriptive statistics, 
visualizations, correlation analysis, and interactive exploration to understand the factors influencing university rankings.

##  Technologies Used

* Python
* Jupyter Notebook
* Pandas
* NumPy
* Matplotlib
* Seaborn

## Analysis Performed

The following analyses were carried out in this project:

* Data loading and exploration
* Checking data types
* Missing value detection and handling
* Statistical summary of numerical columns
* Country-wise university score analysis
* Institution-wise performance analysis
* Average score calculation
* Top 10 university analysis
* Distribution analysis using Histogram
* Outlier detection using Box Plot
* Scatter plot analysis
* Correlation Matrix
* Heatmap visualization
* Pivot Table analysis
* Institution search using user input

##  Overall Insights

The analysis of the Top University dataset provides a comprehensive understanding of the performance 
of universities across the world based on various ranking indicators such as World Rank, Quality of
Education, Alumni Employment, Quality of Faculty, Publications, Influence, Citations, Patents, Broad 
Impact, and Overall Score. After cleaning the dataset by handling missing values in the Broad Impact
column, the data was explored using statistical analysis and visualizations to identify meaningful trends. 
The analysis revealed that **Harvard University** consistently ranked as the top university in the dataset, 
followed by globally recognized institutions such as **Stanford University**, **Massachusetts Institute of
Technology (MIT)**, **University of Cambridge**, and **University of Oxford**, demonstrating their excellence
in education, research, and faculty quality. 

Country-wise analysis showed that the **United States** dominates 
the global university rankings with the highest number of top-ranked universities and the highest overall scores,
while countries such as the **United Kingdom**, **Canada**, **Germany**, and **Australia** also perform strongly
due to their well-established higher education systems. The statistical analysis and visualizations indicate that
only a small number of universities achieve exceptionally high scores, whereas the majority fall within a moderate
performance range. Correlation analysis further reveals a strong positive relationship between publications, citations,
influence, and overall university scores, highlighting that institutions producing high-quality research and receiving 
greater academic recognition tend to achieve better global rankings.

Additionally, universities with higher quality education generally show better alumni
employment outcomes, emphasizing the impact of academic excellence on graduate 
success. The pivot table analysis demonstrates that leading universities maintain consistent 
rankings and scores over multiple years, reflecting their sustained academic performance
and global reputation. Overall, the analysis concludesthat research productivity, citation impact, 
faculty quality, and educational excellence are the primary factors 
contributing to a university's global ranking, with Harvard University emerging as the 
best-performing institution and the United States leading as the most dominant country in higher education.
